# 信号

# 信号

信号是信息的载体或表达形式。

# 信号的频谱

通过傅里叶变换将信号的时域变换到频域的谱线。

# 模拟信号和数字信号

模拟信号：在时间和幅值上都是连续的信号。

数字信号：在时间和幅值上都是离散的信号。

处理模拟信号的电子电路称为模拟电路。模电的任务是不失真地放大信号。

  2. |A|>1，且保持常数

# 信号失真

非线性失真：输出幅值有限而产生的非线性失真。会导致产生新的谐波分量（因为将正弦量变成了非正弦量）。

频率失真：由于线性电抗元件频率响应不同而产生的失真，导致分量信号的比例关系（幅度失真）和时间关系（相位失真）发生变化。不会有新的谐波分量产生。
