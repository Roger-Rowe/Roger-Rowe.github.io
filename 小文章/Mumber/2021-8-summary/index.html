# 身体情况

经历了肾盂输尿管成形术、多次取支架术、软镜碎石术等多次手术，咕了一年的体育课，再加之去年 20 年疫情阶段缺乏运动，身体情况应是不太乐观。经历过多次手术现腹痛出现次数大大减少，这将有利于将后的发展。

对下半年的展望：希望身体 upupup 起来，以下是一些计划

  • [x] 保证不久坐

  • [x] 保证不喝饮料

  • [x] 保证每日 2500ml 饮水量

  • [x] 保证能够好好上体育课

  • [x] 保证每日 30min 的慢跑

  • [x] 保证每日 30min 的力量训练

# 学习情况

经历去年的手术落下 3 门课程，缓考至 21 春季学期，加之寒假没有复习，在开学时的缓考花了大量时间，落下了较多的本学期课程。不仅如此，在毛概、模电、汇编、JAVA 课与计算机系统基础存在大量水课现象。还好后面的考试都是每周一考有较多的复习时间对成绩影响不大。虽然部分课程确实是讲得枯燥，但是要吸取本学期教训，不能因为课堂枯燥就认为课程枯燥，针对这种情况以后可先试听两节课再决定是否继续听或者转而网课学习。另外，由于本学期的排石手术落下了模电考试，务必在暑期将模电学习扎实。两次六级考试均因手术错过，并且英语未系统学习已有两年，英语的系统学习要提上日程。同时做好两手准备，进而考研退可工作，学习内容扎实，专业知识编程能力也要多加练习提高。制定如下计划：

  • [x] 在暑期完成模电复习 ---> 应对开学模电缓考

  • [x] 在暑期完成剑指 offer 练习并开始 Leetcode 每日 3 题的习惯 ---> 学习数据结构与算法，提升编程能力 ---> 工作

  • [x] 在暑期开始系统学习英语并学习大头部分，开学后工作日将花少量时间学习，双休日花较多时间学习英语 ---> 英语积累 ---> 考研英语、六级英语

  • [x] 认真、及时完成课堂任务，在课堂时间做到不水课，课后做到及时总结 ---> 学习内容需要扎实，提高绩点

  • [x] 对于周末要求更高，逐步涉及设计模式的学习并提高工程能力 ---> 工作

# 情感情况

与她的恋爱自 12.20 起，至 5.22 结。有过很快乐的时光，也曾深深地爱过。所以为何短短结束了呢？我想了很久应该是有以下原因：

  1. 不够爱
  2. 兴趣交集太少
  3. 我太忙而你太闲
  4. 成长环境差异太大
  5. 这段感情让双方都累
  6. 我相信未来，可你不确定
  7. 最主要的还是自己过于将具体的人抽象化至脑海中，究竟我喜欢的是抽象出来的她，还是她？

所谓的个人自由不是最终极的追求。我们有时为了美好的事可以做一些退让，那就是情感战胜了自我，获得相对和谐的共生。人们常常过于强调自我，常常会引起争端，而适度地放弃自我，恰恰才能成全自我。

你曾问我什么叫做精神上的爱情，可能就是我精神上的爱情结束了这一切吧。

对于情感，暂时没有期望，未来渺然，谁能陪谁走多远呢？就像你说的我们还有好多年，可我们将不会再见了... 如果有机会再见的话，那再爱吧！希望你曾经也和我一样开心过，不后悔这段感情！祝你前程似锦！祝你 Happy，但希望不是 Happier 哈哈哈～

# 大事件

  • 2021 是党的 100 周年，希望我党不忘初心、永葆青春啊～
  • 2020 东京奥运会推迟到了今年举行，中国健儿们加油（暂时金牌、奖牌数第一喔～）
  • 2021.7 郑州出现了洪灾，灾难无情人有情，中国人向来是团结的，我深深认可这句话
  • 2021.7 吴亦凡爆出性、毒丑闻，希望给予社会公道，真理必定战胜强权
  • 2021.7 疫情开始反弹，希望这轮 delta 新冠快快平息，社会快快运转吧
